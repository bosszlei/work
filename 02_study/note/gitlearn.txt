GitHub Watch按钮：
	Watch可以理解为关注的意思，默认情况下是Not watching，当选择Watch后，你会收到这个GitHub项目的所有动态。
	比如：有人发起pull request或者issue等。接收动态方式包括个人通知中心或者邮箱。
		
GitHub Star按钮：
	Star（星星）可以理解为朋友圈点赞，遇到喜欢的项目可以Star一下，表示对这个项目支持和赞许。
	你Star过的项目可以通过个人头像下的Your stars查看。
		
GitHub Fork按钮：
	某个GitHub项目可能存在bug，或者你想帮助原项目优化一下，这时候你可以使用Fork功能。Fork一下可以拷贝别人的原项目，
	然后你在原项目的基础上优化和完善，然后再发起pull request给原项目作者。
